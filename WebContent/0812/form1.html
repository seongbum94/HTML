입력양식 : form
	데이터를 입력하고 서버로 전송하기 위해서 form이 필요하다
	입력데이터의 형식 : text, password, textarea, select, radio, checkbox
					tel, email, date, month
	전송 방식 : get, post
		get : url 주소 뒤에 ?데이터를 붙여서 전송한다.
		post : request header속에 데이터를 넣어서 전송한다.
