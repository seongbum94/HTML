의사 클래스 (pseudo-class) : 
		-클래스가 정의된 것처럼 간주
		-특정 상태에 따라 속성을 설정
		- 선택자:클래스이름 -> 클래스이름은 . 아니고 : 이다
		-클래스 이름은 바꿀 수 없다.
	a:link{
		color : red;
		text-decoration : none;
	}
	a:visited{
		color : green;
		text-decoration : underline;
	}
	a:hover{
		color : blue;
		text-decoration : underline;
	}
	a:active{
		color : orange;
		font-size : 1.3em;
		text-decoration : overline;
	}
	p:hover{
		font-size : 1.5em;
		background : lightgreen;
	}
	p:active{
		color : white;
	}
