전송기능 -form 태그의 action에 지정된 서버페이지가 실행되어진다.
	- input type ="submit"
	- button type = "submit"
	- input type = "image"
	- button
	
	일반버튼
	- input type = "button"
	- button type = "button"
	- 전송기능이 없다, 스크립트를 통해서 이벤트에 의한 기능을 부여한다.
	
	<input type ="button" value = "확인" onclick="proc()">
	<button type ="button" onclick="proc()">확인<button>
	
	<script>
	function proc(){
		idVal = document.ff.id.value;
		passVal = document.ff.pass.value;
		
		document.ff.action = "f1.jsp?id=" + idVal + "&pass="+ passVal;
		document.ff.submit();
	}
	</script>
