select/option
		- 항목 리스트에서 한개 또는 여러개를 선택할 수 
		- 여러개를 선택시 select태그에서 multiple = "multiple" 속성을 설정한다.
		- 화면에 표시할 항목의 갯수 : select태그에서 size 속성을 설정한다.
		- 초기 실행시 표시될 항목 : option태그에서 selected = "selected" 속성을 설정한다.
		- option태그의 value속성은 서버로 전송되는 값을 설정
		
		input 태그의 value 속성은 초기실행시 화면에 표시되는 문자
		input 태그는 한줄로만 입력이 가능하다
		여러줄을 입력하기 위해선 textarea태그를 사용한다.
		가로길이 : cols 속성
		세로길이  : rows 속성을 사용한다.
